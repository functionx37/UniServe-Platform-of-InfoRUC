# UniServe（人大信息学院综合服务与党团管理平台）使用说明

本文档基于当前仓库代码实现整理，面向「学生端微信小程序」与「老师/管理员 PC 管理后台」的使用方式说明，帮助验收与演示时快速按功能走通流程。

---

## 1. 系统组成与角色

本仓库包含三个部分：

- 学生端（微信小程序）：目录 `miniprogram/`  
  面向学生使用，提供登录、智能问答、党团流程进度、事务申请、成绩单上传与学业分析、通知公告、模板下载、个人中心等功能。
- 管理后台（PC 前端原型）：目录 `admin-frontend/`  
  面向管理老师使用，提供总览、批量导入导出、按标签精准推送、多渠道发送、联调与测试建议等能力。
- 后端服务（Spring Boot）：目录 `backend/`  
  当前代码包含统一返回体 `Result` 与测试接口，可用于验证服务启动与返回体格式。

---

## 2. 快速开始（运行方式）

### 2.1 学生端小程序（微信开发者工具）

1. 打开微信开发者工具 →「导入项目」
2. 选择项目目录：`UniServe-Platform-of-InfoRUC/miniprogram/`
3. 编译运行后即可进入登录页/首页进行功能体验

备注：

- 若需要联调真实后端，请在 `miniprogram/config/env.js` 配置后端地址（`BASE_URL`）并确保后端支持 `Authorization: Bearer <token>` 鉴权头。

### 2.2 管理后台（本地启动）

在 `admin-frontend/` 目录下执行：

```bash
npm install
npm run dev
```

启动后访问：

```text
http://localhost:5173
```

### 2.3 后端服务（Spring Boot）

在 `backend/` 目录下执行：

```bash
./mvnw spring-boot:run
```

启动成功后，可通过测试接口验证服务就绪（端口以日志为准）：

```text
GET /api/test/hello
```

## 3. 学生端小程序使用指南

学生端以底部 TabBar 为主入口：

- 首页：模块入口与待办汇总
- 问答：智能问答
- 事务：事务申请列表
- 我的：个人中心

### 3.1 登录

入口：`登录` 页面。

操作步骤：

1. 输入「学号」与「姓名」
2. 点击「登录」
3. 登录成功后自动进入「首页」

说明：

- 登录成功会写入本地登录态（用于后续页面的登录校验与鉴权请求头）。
- 若未登录直接进入需要登录的页面，会自动跳转到登录页。

### 3.2 首页（待办汇总 + 模块入口）

入口：TabBar「首页」。

页面内容：

- 展示当前登录用户名称
- 展示两个核心待办摘要：
  - 党团待办数量
  - 审批中申请数量
- 提供快捷入口：党团流程、学业分析、通知公告、模板下载、智能问答、事务申请

常见用法：

- 点击「党团流程」进入党团进度时间线
- 点击「学业分析」进入学业概览（可进一步上传成绩单/查看分析结果）
- 点击「通知公告」查看分类通知
- 点击「模板下载」复制预览/下载链接

### 3.3 智能问答

入口：TabBar「问答」。

操作步骤：

1. 在输入框输入问题（或点击「快捷问题」）
2. 点击发送后，页面会新增一条「用户消息」
3. 返回结果后，页面会新增一条「系统回答」

可用交互：

- 相关问题：点击「相关追问」可一键继续提问
- 来源链接：如回答包含来源链接，可点击复制链接（便于在浏览器打开）
- 失败重试：请求失败后可点击重试（按最近一次问题再次请求）

### 3.4 党团流程进度

入口：首页入口 →「党团流程」。

页面内容：

- 党团流程整体进度（百分比）
- 流程节点时间线（已完成/当前/待办）
- 待办事项列表（如有）

使用方式：

- 打开页面会自动刷新当前进度与节点状态
- 若加载失败，可点击重试重新拉取

### 3.5 事务申请（列表 / 筛选 / 新建 / 详情）

#### 3.5.1 申请列表与筛选

入口：TabBar「事务」。

操作步骤：

1. 顶部选择筛选条件：`全部 / 审批中 / 已通过 / 已驳回`
2. 列表会按筛选条件刷新
3. 点击列表项进入「申请详情」

#### 3.5.2 新建申请

入口：事务列表页 →「新建申请」。

支持的申请类型（当前页面内置三类）：

- 请假申请
- 在读证明
- 政治面貌证明

操作步骤：

1. 选择申请类型
2. 按类型填写表单必填项
3. 可选：添加附件（文件选择由组件封装，页面记录附件名称）
4. 点击提交，弹窗展示申请编号，并自动跳转到详情页

必填校验规则（页面侧）：

- 请假申请：请假开始/结束日期、事由、联系电话
- 在读证明：用途说明、接收单位/部门
- 政治面貌证明：用途说明、接收单位/部门

#### 3.5.3 申请详情

入口：事务列表点击某条申请。

页面内容：

- 申请编号、提交时间、状态（用于展示不同颜色态）
- 表单字段展示（不同类型展示不同字段）
- 附件列表（如有，可复制链接/信息）
- 审批时间线与审核意见摘要（如有）
- 结果（如有）：可通过弹窗复制下载链接

隐私与安全：

- 详情页对联系电话等敏感信息做脱敏展示。

### 3.6 学业概览、成绩单上传与学业分析结果

#### 3.6.1 学业概览

入口：首页入口 →「学业分析」。

页面内容：

- 培养方案总学分 / 已完成学分 / 缺少学分
- 完成度百分比
- 上传提示（显示最近上传文件名或提示支持格式）
- 风险提示（显示风险点数量或“暂无风险提示”）

可用操作：

- 点击「上传成绩单」进入上传页
- 点击「查看分析结果」进入分析结果页

#### 3.6.2 成绩单上传

入口：学业概览页 →「上传成绩单」。

支持格式与限制（页面校验）：

- 文件格式：`pdf / xls / xlsx`
- 大小限制：10MB

操作步骤：

1. 选择文件
2. 若格式/大小不符合要求，页面会提示并清空选择
3. 点击「上传」
4. 上传成功后可在弹窗中选择「查看结果」跳转到分析结果页

#### 3.6.3 学业分析结果

入口：学业概览页 →「查看分析结果」或上传成功弹窗 →「查看结果」。

页面内容：

- 总学分 / 已完成 / 缺少
- 模块化进度与建议（按后端返回数据展示）

### 3.7 通知公告

入口：首页入口 →「通知公告」。

操作步骤：

1. 顶部选择分类筛选：`全部 / 就业 / 实习 / 竞赛 / 党团 / 学业`
2. 点击某条通知可展开详情（页面会拉取详情并合并到列表项中）
3. 若通知包含链接，可点击复制链接
4. 首次展开未读通知时会自动标记为已读

### 3.8 模板下载

入口：首页入口 →「模板下载」。

操作步骤：

1. 进入页面后加载模板列表
2. 点击「预览」：生成预览链接并复制到剪贴板（用于在浏览器打开）
3. 点击「下载」：生成下载链接并复制到剪贴板

### 3.9 个人中心（我的）

入口：TabBar「我的」。

页面能力：

- 展示用户基础信息（姓名/学号/学院/年级）
- 展示联系方式（手机号、身份证号）并进行脱敏
- 提供快捷入口：事务申请、成绩单上传
- 退出登录：清理本地登录态并返回首页

## 4. 管理后台（PC）使用指南

管理后台左侧为导航栏，右侧为工作区。主要包含四个视图：

- 总览
- 批量导入导出
- 精准推送
- 联调与测试

### 4.1 总览

入口：左侧导航「总览」。

页面内容：

- 核心指标卡片：
  - 待发布通知数量
  - 当前标签组合命中对象数量
  - 近 7 日发送任务数量（以发送日志统计）
  - 最新导入成功率
- 最近发送记录（展示最新几条日志）
- 快捷操作：
  - 下载导入模板
  - 导出当前通知

### 4.2 批量导入导出（Excel / CSV）

入口：左侧导航「批量导入导出」。

支持文件格式：

- `.xlsx` / `.xls` / `.csv`（读取第一张工作表）

推荐流程：

1. 点击「下载模板」获取示例模板（包含字段列）
2. 按模板字段准备数据
3. 点击上传文件，系统会按约定字段解析并导入
4. 在「导入历史」查看导入批次、总行数、成功行数、失败行数
5. 在「通知数据表」查看导入后的通知记录
6. 如需导出，点击「导出全部数据」

导入字段约定（表头名）：

- 标题、分类、年级、专业、渠道、发布时间、状态

### 4.3 精准推送（按标签筛选 + 多渠道发送）

入口：左侧导航「精准推送」。

操作步骤：

1. 在「标签筛选」区域选择：
   - 年级
   - 专业
   - 身份标签
2. 勾选发送渠道（可多选）：
   - 站内消息 / 邮件 / 导出名单（以页面配置为准）
3. 右侧「命中对象预览」会展示符合条件的学生列表与数量
4. 在「消息草稿」填写标题与正文
5. 点击「发送到匹配对象」
6. 发送后在「发送日志」看到新增的发送记录（包含命中人数与状态）

### 4.4 联调与测试

入口：左侧导航「联调与测试」。

该视图主要提供：

- 建议的接口路径清单（便于后端对齐实现）
- 联调要求（统一返回体结构等）
- 测试建议（Postman 覆盖、压测与边界异常）

## 5. 常见问题（体验与演示）

- 登录后仍被提示未登录：请先退出登录再重新登录；或在微信开发者工具清理缓存后重试。
- 上传按钮不可用：需要先选择文件；文件必须满足格式与大小限制。
- 通知展开失败：请重试展开；若后端联调中，请确认通知详情接口可用且返回结构正确。
- 页面显示为空：属于正常状态之一（空态），可切换筛选条件或检查后端是否返回数据。
