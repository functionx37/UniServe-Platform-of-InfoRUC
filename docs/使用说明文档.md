## 2. 管理后台（PC）使用指南

管理后台左侧为导航栏，右侧为工作区。主要包含四个视图：

- 总览
- 批量导入导出
- 精准推送
- 联调与测试

### 2.1 总览

入口：左侧导航「总览」。

页面内容：

- 核心指标卡片：
  - 待审事务数量
  - 待发布通知数量
  - 当前标签组合命中对象数量
  - 最新导入成功率
- 最近事务申请记录，展示最新几条日志（点击“更多”跳转“事务审批中心”）
- 推送发送记录（点击“更多”跳转“通知公告管理”）

### 2.2 通知公告管理

1. 支持批量导入。
2. 支持对具体条目进行删除。

### 2.3 精准信息推送

入口：左侧导航「精准信息推送」。

操作步骤：

1. 在「标签筛选」区域选择：
   - 年级
   - 专业
   - 身份标签
2. 可以勾选发送渠道（可多选）：
   - 站内消息 / 邮件 / 导出名单
3. 右侧「命中对象预览」会展示符合条件的学生列表与数量
4. 在「消息草稿」填写标题与正文
5. 点击「发送到匹配对象」
6. 发送后在「发送日志」看到新增的发送记录（包含命中人数与状态）

### 2.4 事务审批中心

1. 可对具体的学生事务申请进行审批。
2. 可以对具体条目执行删除操作。
3. 点击详情可以查看每个条目的详细信息。

### 2.5 学生档案管理

1. 支持导入单个学生、批量导入学生。
2. 支持基本的查询与删除操作。
3. 支持依据标签进行筛选。

### 2.6 AI知识库维护

1. 与学生端的“智能政策问答”相关。用于AI索引对应知识库。
2. 可以对知识库进行基本的管理操作，如上传与删除、处理关键词与文档的索引关系、维护与同步。

### 2.7 培养方案管理

1. 上传官方培养方案。
2. 可以进行更新与上传操作。

## 3. 学生端小程序使用指南

学生端以底部 TabBar 为主入口：

- 首页：模块入口与待办汇总
- 问答：智能问答
- 事务：事务申请列表
- 我的：个人中心

### 3.1 登录

入口：`登录` 页面。

操作步骤：

1. 输入「学号」与「姓名」
2. 点击「登录」
3. 登录成功后自动进入「首页」

说明：

- 登录成功会写入本地登录态（用于后续页面的登录校验与鉴权请求头）。
- 若未登录直接进入需要登录的页面，会自动跳转到登录页。

### 3.2 首页（待办汇总 + 模块入口）

入口：TabBar「首页」。

页面内容：

- 展示当前登录用户名称
- 展示两个核心待办摘要：
  - 党团待办数量
  - 审批中申请数量
- 提供快捷入口：党团流程、学业分析、通知公告、模板下载、智能问答、事务申请

常见用法：

- 点击「党团流程」进入党团进度时间线
- 点击「学业分析」进入学业概览（可进一步上传成绩单/查看分析结果）
- 点击「通知公告」查看分类通知
- 点击「模板下载」复制预览/下载链接

### 3.3 智能问答

入口：TabBar「问答」。

操作步骤：

1. 在输入框输入问题（或点击「快捷问题」）
2. 点击发送后，页面会新增一条「用户消息」
3. 返回结果后，页面会新增一条「系统回答」

可用交互：

- 相关问题：点击「相关追问」可一键继续提问
- 来源链接：如回答包含来源链接，可点击复制链接（便于在浏览器打开）
- 失败重试：请求失败后可点击重试（按最近一次问题再次请求）

### 3.4 党团流程进度

入口：首页入口 →「党团流程」。

页面内容：

- 党团流程整体进度（百分比）
- 流程节点时间线（已完成/当前/待办）
- 待办事项列表（如有）

使用方式：

- 打开页面会自动刷新当前进度与节点状态
- 若加载失败，可点击重试重新拉取

### 3.5 事务申请（列表 / 筛选 / 新建 / 详情）

#### 3.5.1 申请列表与筛选

入口：TabBar「事务」。

操作步骤：

1. 顶部选择筛选条件：`全部 / 审批中 / 已通过 / 已驳回`
2. 列表会按筛选条件刷新
3. 点击列表项进入「申请详情」

#### 3.5.2 新建申请

入口：事务列表页 →「新建申请」。

支持的申请类型（当前页面内置三类）：

- 请假申请
- 在读证明
- 政治面貌证明

操作步骤：

1. 选择申请类型
2. 按类型填写表单必填项
3. 可选：添加附件（文件选择由组件封装，页面记录附件名称）
4. 点击提交，弹窗展示申请编号，并自动跳转到详情页

必填校验规则（页面侧）：

- 请假申请：请假开始/结束日期、事由、联系电话
- 在读证明：用途说明、接收单位/部门
- 政治面貌证明：用途说明、接收单位/部门

#### 3.5.3 申请详情

入口：事务列表点击某条申请。

页面内容：

- 申请编号、提交时间、状态（用于展示不同颜色态）
- 表单字段展示（不同类型展示不同字段）
- 附件列表（如有，可复制链接/信息）
- 审批时间线与审核意见摘要（如有）
- 结果（如有）：展示结果文件信息与下载链接

隐私与安全：

- 详情页对联系电话等敏感信息做脱敏展示。
- 若结果文件下载需要鉴权（当前后端证明下载接口要求携带登录 token），在外部浏览器直接打开链接可能会因缺少请求头而失败。验收演示时可先确认后端是否提供“可直接访问的下载链接/签名 URL”，或使用携带 `Authorization` 的方式下载。

### 3.6 学业概览、成绩单上传与学业分析结果

#### 3.6.1 学业概览

入口：首页入口 →「学业分析」。

页面内容：

- 培养方案总学分 / 已完成学分 / 缺少学分
- 完成度百分比
- 上传提示（显示最近上传文件名或提示支持格式）
- 风险提示（显示风险点数量或“暂无风险提示”）

可用操作：

- 点击「上传成绩单」进入上传页
- 点击「查看分析结果」进入分析结果页

#### 3.6.2 成绩单上传

入口：学业概览页 →「上传成绩单」。

支持格式与限制（页面校验）：

- 文件格式：`pdf / xls / xlsx`
- 大小限制：10MB（页面限制）；后端接收上限为 20MB

操作步骤：

1. 选择文件
2. 若格式/大小不符合要求，页面会提示并清空选择
3. 点击「上传」
4. 上传成功后可在弹窗中选择「查看结果」跳转到分析结果页

#### 3.6.3 学业分析结果

入口：学业概览页 →「查看分析结果」或上传成功弹窗 →「查看结果」。

页面内容：

- 总学分 / 已完成 / 缺少
- 模块化进度与建议（按后端返回数据展示）

### 3.7 通知公告

入口：首页入口 →「通知公告」。

操作步骤：

1. 顶部选择分类筛选：`全部 / 就业 / 实习 / 竞赛 / 党团 / 学业`
2. 点击某条通知可展开详情（页面会拉取详情并合并到列表项中）
3. 若通知包含链接，可点击复制链接
4. 首次展开未读通知时会自动标记为已读

### 3.8 模板下载

入口：首页入口 →「模板下载」。

操作步骤：

1. 进入页面后加载模板列表
2. 点击「预览」：生成预览链接并复制到剪贴板（用于在浏览器打开）
3. 点击「下载」：生成下载链接并复制到剪贴板

### 3.9 个人中心（我的）

入口：TabBar「我的」。

页面能力：

- 展示用户基础信息（姓名/学号/学院/年级）
- 展示联系方式（手机号、身份证号）并进行脱敏
- 提供快捷入口：事务申请、成绩单上传
- 退出登录：清理本地登录态并返回首页
