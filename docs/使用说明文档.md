## 2. 管理后台（Web端）使用指南

管理后台左侧为导航栏，右侧为工作区。当前实现包含以下核心模块：

- 工作台总览
- 通知公告管理
- 精准信息推送
- 事务审批中心
- 学生档案中心
- AI 知识库维护
- 培养方案管理

### 2.1 总览

入口：左侧导航「总览」。

页面内容：

- 核心指标卡片：
  - 待审事务数量
  - 待发布通知数量
  - 活跃学生数量
  - 最新导入成功率
- 最近事务申请记录，展示最新几条日志（点击“更多”跳转“事务审批中心”）
- 推送发送记录（点击“更多”跳转“通知公告管理”）
- 最近 5 条操作记录与审计日志：用于解释“为什么导入失败/为什么推送失败/为什么操作被拒绝”等（失败记录会以红色显示错误原因）

### 2.2 通知公告管理

入口：左侧导航「通知公告管理」。

页面能力：

1. 下载导入模板：点击「📄 下载模板」获取 `notifications_import_template.xlsx`（也可上传 CSV/XLSX）。
2. 批量导入：点击「📤 批量导入」，选择文件后系统会解析并导入到通知列表。
3. 发布/下线：对每条通知可执行「发布/下线」切换（对应通知的状态）。
4. 删除：对单条通知可执行删除。

导入字段（与当前前端解析逻辑一致，Excel 表头可使用中文或英文别名）：

- 标题（title）
- 分类（category）
- 年级（grade，默认“全部”）
- 专业（major，默认“全部”）
- 渠道（channel，默认“站内消息”）
- 状态（status，默认“待发布”）
- 内容（content）
- 链接（links，建议为 JSON 数组字符串，例如 `[]` / `["https://..."]`）

说明：

- 若分类为「推送 / 精准推送」，并且状态为「已发布」，系统会同步生成一条推送发送记录（用于追溯）。
- 导入成功/失败的汇总会弹窗提示；如果存在失败行，请到「总览 → 最近操作记录与审计日志」查看失败原因。

### 2.3 精准信息推送

入口：左侧导航「精准信息推送」。

操作步骤：

1. 在「标签筛选」区域选择：
   - 年级
   - 专业
   - 身份标签
2. 可以勾选发送渠道（可多选）：
   - 站内消息 / 邮件 / 导出名单
3. 右侧「命中对象预览」会展示符合条件的学生列表与数量
4. 在「消息草稿」填写标题与正文
5. 点击「发送到匹配对象」
6. 发送后在「发送日志」看到新增的发送记录（包含命中人数与状态）

补充说明（与当前实现逻辑一致）：

- 若当前筛选条件命中人数为 0，系统不会抛出错误，但会记录一条状态为「无法推送」的发送日志，便于排查“为什么没有任何人收到”。

### 2.4 事务审批中心

入口：左侧导航「事务审批中心」。

页面能力：

1. 列表查看：按状态查看学生提交的事务申请。
2. 详情查看：点击「详情」打开弹窗，查看表单内容、时间线、附件材料等。
3. 审批操作：对状态为「待审核/审批中」的申请可执行「通过审批 / 驳回申请」并可填写审批意见。
4. 附件下载：若申请包含附件材料，详情弹窗的「附件材料」区域可直接点击下载（用于查看学生提交的材料文件）。
5. 删除记录：可删除某条申请记录（谨慎使用）。

### 2.5 学生档案管理

入口：左侧导航「学生档案中心」。

页面能力：

1. 单个录入：点击「➕ 录入学生」打开录入弹窗，保存后学生即可用“学号 + 姓名”登录小程序。
2. 下载导入模板：点击「📄 下载模板」获取 `users_import_template.xlsx`（也可上传 CSV/XLSX）。
3. 批量导入：点击「📥 批量导入」选择文件导入学生档案。
4. 查询与筛选：支持按姓名/学号/账号搜索，按角色/年级/专业筛选。
5. 详情查看：点击「详情」查看学生档案字段。

导入字段（与当前前端解析逻辑一致，Excel 表头可使用中文或英文别名）：

- 学号/用户名（学号会同时作为默认用户名使用）
- 姓名
- 年级、专业、身份
- 邮箱、手机号（可空）
- 角色/角色ID：支持 “普通学生/班团骨干” 或 4/3（系统会过滤掉非学生与非骨干数据）

说明：

- 导入完成会提示“成功/失败行数”。若存在失败行，请到「总览 → 最近操作记录与审计日志」查看失败原因。

### 2.6 AI知识库维护

1. 与学生端的“智能政策问答”相关。用于AI索引对应知识库。
2. 可以对知识库进行基本的管理操作，如上传与删除、处理关键词与文档的索引关系、维护与同步。

### 2.7 培养方案管理

入口：左侧导航「培养方案管理」。

页面能力：

1. 下载课程模板：点击「📄 下载课程模板」获取 `courses_import_template.xlsx`。
2. 上传培养方案：上传 Excel 后系统解析课程/模块数据，供学生端学业分析使用。
3. 删除与替换：可删除当前方案并重新上传。

## 3. 学生端小程序使用指南

学生端以底部 TabBar 为主入口：

- 首页：模块入口与待办汇总
- 问答：智能问答
- 事务：事务申请列表
- 我的：个人中心

### 3.1 登录

入口：`登录` 页面。

操作步骤：

1. 输入「学号」与「姓名」
2. 点击「登录」
3. 登录成功后自动进入「首页」

说明：

- 登录成功会写入本地登录态（用于后续页面的登录校验与鉴权请求头）。
- 若未登录直接进入需要登录的页面，会自动跳转到登录页。

### 3.2 首页（待办汇总 + 模块入口）

入口：TabBar「首页」。

页面内容：

- 展示当前登录用户名称
- 展示两个核心待办摘要：
  - 党团待办数量
  - 审批中申请数量
- 提供快捷入口：党团流程、学业分析、通知公告、模板下载、智能问答、事务申请

常见用法：

- 点击「党团流程」进入党团进度时间线
- 点击「学业分析」进入学业概览（可进一步上传成绩单/查看分析结果）
- 点击「通知公告」查看分类通知
- 点击「模板下载」复制预览/下载链接

### 3.3 智能问答

入口：TabBar「问答」。

操作步骤：

1. 在输入框输入问题（或点击「快捷问题」）
2. 点击发送后，页面会新增一条「用户消息」
3. 返回结果后，页面会新增一条「系统回答」

可用交互：

- 相关问题：点击「相关追问」可一键继续提问
- 来源链接：如回答包含来源链接，可点击复制链接（便于在浏览器打开）
- 失败重试：请求失败后可点击重试（按最近一次问题再次请求）

### 3.4 党团流程进度

入口：首页入口 →「党团流程」。

页面内容：

- 党团流程整体进度（百分比）
- 流程节点时间线（已完成/当前/待办）
- 待办事项列表（如有）

使用方式：

- 打开页面会自动刷新当前进度与节点状态
- 若加载失败，可点击重试重新拉取

### 3.5 事务申请（列表 / 筛选 / 新建 / 详情）

#### 3.5.1 申请列表与筛选

入口：TabBar「事务」。

操作步骤：

1. 顶部选择筛选条件：`全部 / 审批中 / 已通过 / 已驳回`
2. 列表会按筛选条件刷新
3. 点击列表项进入「申请详情」

#### 3.5.2 新建申请

入口：事务列表页 →「新建申请」。

支持的申请类型（当前页面内置三类）：

- 请假申请
- 在读证明
- 政治面貌证明

操作步骤：

1. 选择申请类型
2. 按类型填写表单必填项
3. 可选：添加附件材料（提交时会先上传附件，再将附件下载地址写入申请记录，供后续审批下载核验）
4. 点击提交，弹窗展示申请编号，并自动跳转到详情页

必填校验规则（页面侧）：

- 请假申请：请假开始/结束日期、事由、联系电话
- 在读证明：用途说明、接收单位/部门
- 政治面貌证明：用途说明、接收单位/部门

#### 3.5.3 申请详情

入口：事务列表点击某条申请。

页面内容：

- 申请编号、提交时间、状态（用于展示不同颜色态）
- 表单字段展示（不同类型展示不同字段）
- 附件列表（如有，可复制链接/信息）
- 审批时间线与审核意见摘要（如有）
- 结果（如有）：展示结果文件信息与下载链接

隐私与安全：

- 详情页对联系电话等敏感信息做脱敏展示。
- 结果文件（证明 PDF）下载接口需要鉴权（后端会校验当前登录用户是否有权限下载）。如需在浏览器直接下载，请确认下载请求能携带登录态（或由后端提供签名/免鉴权下载方案）。

### 3.6 学业概览、成绩单上传与学业分析结果

#### 3.6.1 学业概览

入口：首页入口 →「学业分析」。

页面内容：

- 培养方案总学分 / 已完成学分 / 缺少学分
- 完成度百分比
- 上传提示（显示最近上传文件名或提示支持格式）
- 风险提示（显示风险点数量或“暂无风险提示”）

可用操作：

- 点击「上传成绩单」进入上传页
- 点击「查看分析结果」进入分析结果页

#### 3.6.2 成绩单上传

入口：学业概览页 →「上传成绩单」。

支持格式与限制（页面校验）：

- 文件格式：`pdf / xls / xlsx`
- 大小限制：10MB（页面限制）；后端接收上限为 20MB

操作步骤：

1. 选择文件
2. 若格式/大小不符合要求，页面会提示并清空选择
3. 点击「上传」
4. 上传成功后可在弹窗中选择「查看结果」跳转到分析结果页

#### 3.6.3 学业分析结果

入口：学业概览页 →「查看分析结果」或上传成功弹窗 →「查看结果」。

页面内容：

- 总学分 / 已完成 / 缺少
- 模块化进度与建议（按后端返回数据展示）

### 3.7 通知公告

入口：首页入口 →「通知公告」。

操作步骤：

1. 顶部选择分类筛选：`全部 / 就业 / 实习 / 竞赛 / 党团 / 学业`
2. 点击某条通知可展开详情（页面会拉取详情并合并到列表项中）
3. 若通知包含链接，可点击复制链接
4. 首次展开未读通知时会自动标记为已读

### 3.8 模板下载

入口：首页入口 →「模板下载」。

操作步骤：

1. 进入页面后加载模板列表
2. 点击「预览」：生成预览链接并复制到剪贴板（用于在浏览器打开）
3. 点击「下载」：生成下载链接并复制到剪贴板

### 3.9 个人中心（我的）

入口：TabBar「我的」。

页面能力：

- 展示用户基础信息（姓名/学号/学院/年级）
- 展示联系方式（手机号、身份证号）并进行脱敏
- 提供快捷入口：事务申请、成绩单上传
- 退出登录：清理本地登录态并返回首页
